# Adding another phenomenon

1. Open `src/data/phenomena.ts`.
2. Add one new object to `PHENOMENA` with a unique `id` and URL-safe `slug`.
3. Complete every field, including two or more authoritative sources, safety wording, quiz answers and related slugs.
4. The directory card, filters, static detail route, metadata, next/previous navigation and quiz are generated automatically.
5. Run `npm run lint`, `npm run typecheck`, `npm run test` and `npm run build` before publishing.

All current hero images and diagrams are original responsive SVG artwork generated inside the site. Credit: EduVerse original illustration.
