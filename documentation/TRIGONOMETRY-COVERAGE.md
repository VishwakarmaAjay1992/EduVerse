# Complete Trigonometry Course — Internal Coverage Checklist

This checklist is an internal development and QA document. It is not linked from the student-facing website.

## Source structure reviewed

The source curriculum contains eight chapters and twenty-six instructional sections. Each instructional section has been assigned to one native EduVerse lesson rather than being divided by printed page.

## Chapter-to-lesson mapping

### 1. Foundations of Trigonometry

- [x] Degree and Radian Measure of Angles
  - Revolutions, degree measure, radian measure, directed angles, conversions, standard position, coterminal angles, complementary and supplementary angles
- [x] Right Triangle Trigonometry
  - Similar triangles, sine/cosine/tangent ratios, special triangles, inverse ratios, right-triangle applications
- [x] The Unit Circle
  - Oriented arcs, coordinate definitions, exact standard values, reference angles, signs, Pythagorean identity
- [x] The Six Trigonometric Functions
  - Reciprocal and quotient functions, domains, signs, exact evaluation, elementary equations
- [x] Foundational Trigonometric Identities
  - Reciprocal, quotient, and Pythagorean identities; simplification and verification
- [x] Beyond the Unit Circle
  - General-coordinate definitions, radius normalization, circle parametrization

### 2. Trigonometric Graphs and Applications of Radian Measure

- [x] Graphs of Sine and Cosine Functions
  - Fundamental cycles, symmetry, amplitude, period, phase shift, vertical shift, equations from graphs
- [x] Graphs of the Other Trigonometric Functions
  - Tangent, cotangent, secant, cosecant, domains, ranges, asymptotes, transformations
- [x] Applications of Radian Measure
  - Arc length, sector area, angular speed, linear speed, rotational applications

### 3. Trigonometric Identities

- [x] Using Trigonometric Identities
  - Even/odd, sum/difference, tangent addition, cofunction identities, exact values, verification
- [x] Multiple-Angle Identities
  - Double-angle, half-angle, power-reduction, product-to-sum, sum-to-product

### 4. Trigonometric Equations

- [x] Inverse Sine and Cosine Functions
  - Restricted domains, principal ranges, exact values, compositions
- [x] The Other Inverse Trigonometric Functions
  - Inverse tangent/cotangent/secant/cosecant, principal values, triangle method
- [x] Inverse Functions and Trigonometric Equations
  - Reference angles, interval solutions, general solutions, transformed equations
- [x] Solving General Trigonometric Equations
  - Factoring, identities, quadratic substitution, multiple angles, validation

### 5. Beyond Right Triangles

- [x] The Law of Sines
  - ASA/AAS, SSA ambiguous case, triangle area, bearings and indirect measurement
- [x] The Law of Cosines
  - SAS/SSS, Pythagorean connection, Heron's formula, applications

### 6. Polar Coordinates and Applications

- [x] Polar Coordinates and Equations
  - Polar points, equivalent coordinates, point conversion, equation conversion
- [x] Graphing Polar Equations
  - Symmetry, circles, limacons/cardioids, roses, lemniscates
- [x] Polar Representations of Complex Numbers
  - Complex plane, modulus, argument, trigonometric and exponential form
- [x] Complex Products, Powers, Quotients, and Roots
  - Polar operations, De Moivre's theorem, powers, complete root sets

### 7. Vectors

- [x] Vector Properties and Operations
  - Scalars/vectors, components, geometric and algebraic operations, magnitude, direction
- [x] The Unit Vector and Vector Applications
  - Normalization, principal unit vectors, forces, velocity, navigation
- [x] The Dot Product
  - Component/geometric definitions, angles, orthogonality, projections, work

### 8. Parametric Equations

- [x] Sketching Curves Described by Parametric Equations
  - Tables, elimination, restrictions, orientation, common curves
- [x] Parametric Descriptions for Oriented Curves
  - Direction/speed changes, line segments, circles, cycloids, projectile motion
